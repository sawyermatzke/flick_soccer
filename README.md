# binho-ios

An iOS multiplayer tabletop flick-soccer game project targeting a faithful digital recreation of Binho-style gameplay, with a branding system that can swap to a generic/original identity if needed.

## Current status

Project is in research/specification and preproduction.

See `docs/STATUS.md` for current milestone and next scheduled work.

## High-level goals

- Build a real App Store-targeted iOS game.
- Deliver live multiplayer in the MVP.
- Keep gameplay/rules engine separate from branding and themed assets.
- Preserve fidelity where possible without blindly copying unverified physical-layout assumptions.

## Principles

- Do not assume board layout from memory.
- Treat public rules/layout evidence as provisional until documented in specs.
- Keep branded assets and copy isolated from core logic.
- Write specs first, then implement against them.
